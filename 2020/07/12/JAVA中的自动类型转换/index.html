JAVA中的自动类型转换

2020-07-12 JAVA 0 评论 字数统计: 82(字) 阅读时长: 1(分)

java中的数据类型自动转换实质上是提升了长度，因此自动转换只能是从长度低的类型转换为长度高的类型
故 byte、short、char可以转换为int类型
int类型可以转换为long、float、double类型
故char+int结果为int类型
char + int + double + “” 结果为String类型
